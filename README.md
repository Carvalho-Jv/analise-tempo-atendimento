# Análise de Tempo de Atendimento

Projeto de análise de dados utilizando Python e Pandas.

## Objetivo
Analisar tempos de atendimento para identificar eficiência operacional e possíveis melhorias no processo.

## Tecnologias utilizadas
- Python
- Pandas

## Métricas analisadas
- Tempo médio de atendimento
- Tempo máximo
- Tempo mínimo
- Avaliação de eficiência do processo

## Exemplo de resultado
O script identifica se o processo está dentro do esperado ou se precisa de melhoria.
